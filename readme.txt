Visitor Management System for WordPress

== Description ==

The **VMS Plugin** provides a complete Visitor Management System for WordPress.  
It allows clubs, offices, and organizations to manage guest visits, host assignments, and entry/exit logs.

Key Features:
* Register guests and assign them to hosts
* Track visit dates and statuses (approved, cancelled, signed in/out)
* Enforce host visitor limits
* Generate visitor reports

Perfect for organizations that want a structured, digital way of handling guests.

== Frequently Asked Questions ==

= Who is this plugin for? =
This plugin is designed for clubs, offices, and organizations that want to streamline visitor handling.

= Does it support custom roles? =
Yes. Hosts and admins can be mapped to roles in WordPress.

= Can I extend it? =
Yes. It uses standard WordPress APIs and can be extended by developers.
